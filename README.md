# Little Talkers — متحدثون صغار (MVP)

تطبيق ويب بسيط وأنيق لمتابعة جلسات علاج النطق والتخاطب، بدورين: **الأخصائي** و **ولي الأمر**.

## التشغيل
لا يحتاج أي تثبيت أو خادم. افتح ملف **`index.html`** مباشرة في المتصفح.

> البيانات تُحفظ محلياً في المتصفح (`localStorage`). لإعادة التهيئة بالبيانات التجريبية شغّل في الـ console: `LittleTalkersReset()`.

## المزايا
**الأخصائي:** إضافة/تعديل/**حذف** المتعالجين، تعيين الطبيب المعالج، كتابة ملاحظات الجلسات، رفع التقارير،
تحديث التقدم، كتابة الخطة القادمة، **جدولة الجلسات القادمة**، استقبال **رسائل أولياء الأمور** (مع عدّاد على جرس الإشعارات).

**الطبيب الرئيسي فقط:** إنشاء/حذف حسابات أطباء آخرين من صفحة «الأطباء».

**سجل النشاط:** كل حدث يُسجّل بالتاريخ واسم الطبيب، وقابل للنقر للانتقال لملف المتعالج، مع فلترة حسب الطبيب.

**ولي الأمر (قراءة فقط):** عرض ملف طفله، التقدم، الملاحظات، الجلسات القادمة، الملفات، الخطة —
وإرسال رسالة للأخصائي (اسم الطبيب + الموضوع + المشكلة/السؤال) ومتابعة حالة القراءة.

**فلاتر المتعالجين:** بحث + حسب الطبيب + حسب التشخيص + ترتيب (الأحدث/التقدم/الاسم).

**بطاقات الإحصائيات** قابلة للنقر وتنقلك للبيانات الصحيحة.
